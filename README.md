Mushaf Al-Tahajjud - Image Edition
🌟 نبذة عن المشروع / About the Project
العربية:
هذا المستودع يحتوي على نسخة مصورة كاملة من مصحف التهجد (180 صفحة). تم إعداد هذا المشروع لتسهيل الوصول إلى صفحات المصحف بجودة عالية واستخدامها في التطبيقات الإسلامية أو القراءة المباشرة.
English:
This repository contains a complete digital image set of Mushaf Al-Tahajjud (180 pages). This project is designed to provide high-quality access to the Mushaf pages for use in Islamic applications or direct reading.
📁 محتويات المستودع / Repository Content
Images Folder: يحتوي على كافة الصفحات بصيغة (PNG/JPG) مرقمة تسلسلياً.
High Quality: الصور واضحة ومناسبة للقراءة الرقمية.
Easy Navigation: تسمية الملفات تبدأ من 001 إلى 180 لضمان الترتيب الصحيح.
🚀 كيفية الاستخدام / How to Use
التحميل (Download): يمكنك تحميل المستودع كاملاً كملف المضغوط (ZIP) من زر Code.
التصفح (Browsing): يمكنك الدخول إلى مجلد الصور وتصفح الصفحات مباشرة من متصفحك.
المساهمة (Contribution): نرحب بأي تحسينات على جودة الصور أو تنظيم الملفات.
